# Machine Learning Journey: From Linear Regression to RNNs

Note: The notebooks and comments in this repository are written in Spanish.

This repository documents my first steps and evolution in the field of Machine Learning and Artificial Intelligence. It showcases a progression from basic statistical models to the implementation of neural networks using both raw NumPy and professional frameworks like TensorFlow/Keras.

## About this Project

I created this repository to track my growth as I explored the foundations of AI. It reflects my passion for understanding how models learn—starting from the basic math of a single neuron to the complexity of recurrent structures. I believe in learning by doing, and these notebooks represent my hands-on approach to mastering ML concepts.

## Roadmap & Contents

The projects are organized by complexity and topic:

### 1. Fundamentals & Manual Implementations
* **Linear Regression (`ML_linear_regression.ipynb`):** A study on the basics of predictive modeling and cost functions.
* **The Perceptron (`Perceptron.ipynb`):** Building a single-neuron model from scratch to understand weights, bias, and activation functions.
* **Neural Network from Scratch (`Red Neuronal Numpy.ipynb`):** Implementing a multi-layer network using only NumPy to master backpropagation and gradient descent without high-level libraries.

### 2. Deep Learning with TensorFlow
* **Cancer Prediction (`Cáncer_Tensorflow.ipynb`):** A binary classification model built with TensorFlow to predict whether a tumor is malignant or benign based on real medical data.
* **Text Generation RNN (`Text_Generator_RNN.ipynb`):** An introduction to Recurrent Neural Networks (RNNs) for sequence modeling, capable of generating text character by character.

## Key Technologies
* **Python 3.x**
* **NumPy:** For manual matrix operations and building models from scratch.
* **TensorFlow / Keras:** For deep learning and production-ready architectures.
* **Scikit-learn:** For data splitting and preprocessing.
* **Pandas & Matplotlib:** For data management and visualization.
